## Quick Start | 快速开始

```bash
# Build | 编译
mkdir build && cd build && cmake .. && make -j$(nproc)

# Run | 运行  
./neurocompass_motion your_4d_data.nii.gz

# Test Data | 测试数据
../scripts/download_test_data.sh
```
